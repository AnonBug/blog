# 多结果分类器：Softmax 回归

## 1. 多结果情形

Logitic 回归输出结果为 0 / 1，当结果有多个可能性时，就不适用了。

![](./imgs/多分类回归模型.png)

## 1.1 Softmax 回归

假设，训练得到了四个数据，需要通过这四个数值映射到分类结果，一种思路就是将最大值置为 1，其它都看作 0。这种映射方式太生硬。

相对的，Softmax 方法本质上就是使这个过程更加温和（Soft）。
